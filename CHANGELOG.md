# Changelog

## Unreleased

### Added

- Add pre-commit hooks for Ruff formatting, autoflake, uv lockfile checks, and
  third-party license generation.
- Add `THIRDPARTYLICENSES` generated by the pre-commit license hook.
- Add regression tests for tmux popup geometry and popup cancellation.

### Fixed

- Position the borderless tmux popup over the target pane instead of using a
  full-client popup, avoiding one-line overlay shifts when tmux status lines
  affect pane placement.
- Treat `Esc` and `Ctrl-c` cancellation as a successful exit so tmux does not
  report a failed `run-shell` command.

### Changed

- Document the pre-commit install and run commands in the development workflow.
- Apply Ruff formatting to the existing Python sources and tests.
